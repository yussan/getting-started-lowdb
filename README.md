# getting-started-lowdb
We are lowdb, Lowdb is a lightweight, type-safe local JSON database built for Node.js, Electron, and browser environments.

## About
Dummy project built with nest js.

## How to Start
### Development Mode
```
pnpm start:dev
```

### Production Mode
```
pnpm start
```

by default program run on port 3000,`http://localhost:3000`
